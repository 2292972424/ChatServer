# ChatSever
可以工作在nginx tcp负载均衡中的集群聊天服务器和客户端源码 基于muduo库 redis json mysql

# 编译方式
下载源码后，执行./build.sh即可

# 自动生成mysql表结构
执行./chat.sql即可
